![Playwright Tests](https://github.com/decoNTANK/Dashboard/actions/workflows/playwright.yml/badge.svg)

## QA Dashboard




A full-stack QA automation portfolio project featuring a bug tracking dashboard with a REST API backend.

The project demonstrates automated testing with Playwright and Jest, API testing, accessibility checks, CI/CD with GitHub Actions, and OpenAPI documentation.
